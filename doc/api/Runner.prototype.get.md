## NAME

`amok.Runner.prototype.get` -- returns the specified setting.

## SYNOPSIS

```js
get(key)
```

## PARAMETERS
`key` *String*
:   The key of the setting to return

## RETURN VALUE

Returns the setting associated with the specified key or undefined if the key
can't be found.

## DESCRIPTION

Returns the value
